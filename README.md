# Pizza ONE 🍕

O **Pizza ONE** é uma aplicação web completa para uma pizzaria fictícia, desenvolvida para gerenciar pedidos, clientes e produtos através de um sistema robusto de back-end.

## 📋 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de sistemas gerenciadores de bancos de dados e integração server-side. A aplicação permite realizar todas as operações fundamentais de um sistema de gestão (CRUD), conectando uma interface amigável a um banco de dados persistente.

## 🛠️ Tecnologias Utilizadas

* **PHP**: Responsável pela lógica de negócio e processamento server-side.
* **SQL (MySQL/MariaDB)**: Estrutura de banco de dados para armazenamento de pedidos e produtos.
* **HTML5 & CSS3**: Estruturação e estilização da interface.
* **JavaScript**: Validações de formulários e interatividade no front-end.

## ⚙️ Funcionalidades (CRUD)

O sistema conta com um fluxo completo de gerenciamento:
* **Create (Criação):** Cadastro de novos clientes e registro de pedidos de pizza.
* **Read (Leitura):** Listagem de cardápio e consulta de pedidos realizados no banco de dados.
* **Update (Atualização):** Alteração de dados cadastrais ou status de pedidos.
* **Delete (Exclusão):** Remoção de registros do sistema.

## 🗄️ Estrutura do Banco de Dados

A aplicação utiliza uma estrutura relacional para garantir a integridade dos dados, incluindo:
* Tabelas de Clientes.
* Tabela de Produtos (Pizzas/Bebidas).
* Tabela de Pedidos com relacionamento entre as entidades.

## 🚀 Como Executar

1.  Clone este repositório.
2.  Certifique-se de ter um servidor local (como XAMPP ou WAMP) configurado.
3.  Importe o arquivo `.sql` disponível na pasta `database` para o seu gerenciador de banco de dados.
4.  Configure as credenciais de acesso no arquivo de conexão PHP.
5.  Acesse `localhost/pizza-one` no seu navegador.
